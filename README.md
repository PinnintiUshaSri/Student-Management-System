# Student Management System

## Project Summary
The Student Management System is a Python-based application designed to manage and analyze student academic records efficiently. It integrates SQLite for database management and uses Pandas and Matplotlib for data analysis and visualization.

This project demonstrates real-world CRUD operations along with reporting and graphical insights.

---
## Project Type
Personal Portfolio Project | Python Data Management System | Data Analysis Application

---

## Tech Stack
- Python (Core Programming)
- SQLite (Database Management)
- Pandas (Data Processing)
- Matplotlib (Data Visualization)

---

## Key Functionalities

### Database Operations (CRUD)
- Add new student records
- View all student data
- Search student by ID
- Update student marks
- Delete student records

### Data Analysis Features
- Identify topper student
- Export dataset to Excel file

### Data Visualization
- Bar chart of student marks distribution
- Pie chart of marks percentage distribution

---

## Project Structure
- `student.py` → Main application with menu-driven system
- `students.db` → SQLite database storing records
- `students_report.xlsx` → Exported dataset
- `marks_graph.png` → Bar chart visualization
- `marks_pie_chart.png` → Pie chart visualization

---

## Installation & Execution

### Install Dependencies

You need to install required Python libraries before running the project.

#### Simple Method:
```
pip install pandas matplotlib openpyxl
```

#### Professional Method
```
Create requirements.txt:
pandas
matplotlib
openpyxl
```
#### Install dependencies:
```
pip install -r requirements.txt
```
#### Run the Project
```
python student.py
```
--- 
## Output Description
- Student data stored in SQLite database (students.db) 
- Excel report generated automatically (students_report.xlsx)
- Charts generated:
  - Bar chart → marks_graph.png
  - Pie chart → marks_pie_chart.png
